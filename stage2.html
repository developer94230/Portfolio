Présentation de l'entreprise

J'ai effectué mon stage au Centre Socio-Culturel La Plaine, association à but non lucratif située au 1, allée Pierre de Montreuil à Cachan. Ce centre est fédéré à l'AGAESC, qui possède également trois autres centres dans la ville.

Le centre a pour mission de favoriser le lien social et propose des activités culturelles, éducatives et de loisirs aux habitants : ateliers, événements, accompagnements, projets collectifs. Il souhaitait améliorer sa présence numérique pour mieux communiquer avec son public.

Projet IA — Automatisation Instagram

Le script PHP surveille un dossier local, génère automatiquement une légende via Hugging Face (Flan-T5) pour chaque photo détectée, puis la publie sur Instagram via l'Instagram Graph API. Ngrok crée une URL publique temporaire pour rendre les images accessibles.

PHP — Simulation publication Instagram
<?php
// Clé Hugging Face
$apiKey = "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";

// Fonction pour générer une légende
function generer_legende($message, $apiKey) {
    $data = ["inputs" => $message];
    $ch = curl_init("https://router.huggingface.co/models/google/flan-t5-small");
    curl_setopt($ch, CURLOPT_HTTPHEADER, [
        "Authorization: Bearer $apiKey",
        "Content-Type: application/json"
    ]);
    curl_setopt($ch, CURLOPT_POST, true);
    curl_setopt($ch, CURLOPT_POSTFIELDS, json_encode($data));
    curl_setopt($ch, CURLOPT_RETURNTRANSFER, true);
    $result = json_decode(curl_exec($ch), true);
    curl_close($ch);

    return $result[0]["generated_text"] ?? "Photo du centre";
}

// Exemple : traiter une image
$image = "exemple.jpg";
$legende = generer_legende("Écris une légende pour cette photo : exemple.jpg", $apiKey);

// Simulation de publication Instagram
file_put_contents("simulation_exemple.txt", "Image : exemple.jpg\nLégende : " . $legende);

Difficultés rencontrées
⛔ Hébergement Laravel

GitHub Pages incompatible — Le site Laravel utilise PHP et MySQL, non supportés par GitHub Pages. Railway a également échoué lors du déploiement.

✅ Solution envisagée

Utiliser un hébergement web compatible PHP/Laravel (ex. o2switch, PlanetHoster) pour déployer le site complet avec base de données.

⛔ URL locale refusée

Instagram Graph API exige des images accessibles via une URL HTTPS publique. Le serveur local localhost est refusé.

✅ Solution

Utilisation de Ngrok pour créer un tunnel HTTPS temporaire vers le dossier local, rendant les images accessibles à l'API.

⛔ Vérification Meta refusée

La vérification d'entreprise Meta est obligatoire pour le mode live. Le document soumis pour le CSC La Plaine a été refusé lors du contrôle.

⏳ En cours

Une nouvelle soumission de documents officiels est prévue pour finaliser la vérification et activer complètement la publication automatique.
