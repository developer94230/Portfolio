Contexte

Centre Socio-Culturel situé à Cachan, fédéré à l'AGAESC. Sa mission : favoriser le lien social via des activités culturelles, éducatives et de loisirs.

Besoin identifié : améliorer la présence numérique du centre pour mieux communiquer avec ses habitants.

🚀
Les deux projets
Projet 01 · Laravel
Site web vitrine

Site complet avec pages publiques (accueil, activités, actualités, adhésion, contact) et portail administrateur sécurisé permettant le CRUD via Blade.

Laravel MVC PHP MySQL HTML/CSS GLightbox @auth / @guest
Projet 02 · IA & API
Automatisation Instagram

Script PHP qui détecte des photos locales, génère une légende via Flan-T5 (Hugging Face), puis publie sur Instagram via l'API Graph de Meta.

PHP Graph API Hugging Face Ngrok

Difficultés rencontrées
⛔ Hébergement Laravel

GitHub Pages & Railway incompatibles — Laravel nécessite PHP + MySQL, non supportés.

✅ Solution envisagée

Hébergement dédié PHP/Laravel (o2switch, PlanetHoster) avec base de données.

⛔ URL locale refusée

Instagram Graph API exige une URL HTTPS publique — localhost refusé.

✅ Solution

Ngrok : tunnel HTTPS temporaire vers le dossier local pour rendre les images accessibles.

⛔ Vérification Meta refusée

La vérification d'entreprise est obligatoire pour le mode live. Document soumis refusé.

⏳ En cours

Nouvelle soumission de documents officiels prévue pour activer la publication automatique.
